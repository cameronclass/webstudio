О PYTHON

Как появился

Python был разработан в конце 1989г. Гуидо ван Россумом во время рождественских каникул, когда его исследовательская лаборатория была закрыта. Он позаимствовал многие средства программирования, присущие другим языкам.

Функционал

Python поддерживает классы, модули, обработку исключений, а также многопоточные вычисления. Язык поддерживает несколько парадигм программирования: структурное, объектно-ориентированное, функциональное и аспектно-ориентированное.

Надежность

Когда дело касается обработки и вычисления большого количества данных, то тут Python – незаменим. Именно это привлекло крупные компании использовать этот язык в своих сервисах: Поиск google, Youtube, NASA. Используется для работы с искусственным интеллектом.
